# 🚀 CareerIQ 

## 📌 About the Project
**CareerIQ** is an AI-powered web application designed for B.Tech CSE students in India to simplify career decisions.

It helps students:
- Analyze their resume
- Identify skill gaps
- Match with suitable job roles
- Find real-time jobs & internships
- Detect fake job scams

---

## 📖 Overview
CareerIQ processes resumes and provides intelligent career insights.

The system works by:
- Extracting text from resumes using PDF.js  
- Matching skills with a predefined skill database (238 skills)  
- Comparing skills with 11 job roles  
- Fetching real-time job listings using an API  
- Filtering opportunities for India + CSE students  
- Detecting fake jobs using pattern analysis  

💡 The platform focuses on providing:
- Fast analysis  
- Relevant opportunities  
- Reliable results  

---

## ✨ Features

- 📄 **Resume Analysis**  
  Extracts and categorizes skills automatically  

- 🎯 **Job Role Matching**  
  Matches user skills with 11 predefined roles  

- 📊 **Skill Gap Analysis**  
  Suggests missing skills and improvement plan  

- 🌐 **Real-Time Job Listings**  
  Fetches live jobs using API  

- 🇮🇳 **India-Specific Filtering**  
  Shows only relevant opportunities  

- 🚨 **Fake Job Detector**  
  Detects scams using domain checks & patterns  

- 📈 **Career Guidance**  
  Provides roadmap and suggestions
